# karrot_market
[넘블 첼린지 - 초기 당근마켓 서비스 구축하기]

## 사용기술
- **Language** - Java11
- **Framework** -  Spring Boot 2.6.2, Spring Data JPA, Spring Security, Tymeleaf
- **Database** : Mysql, H2
- **Infra** : AWS(EC2, RDS, S3)
- **CI**/**CD** : Github Actions, Code Deploy
- **ETC** - QueryDSL

## 담당역할
- API 개발 & 테스트코드 작성
    - 사용자 관련 API
    - 상품 관련 API
- 페이지 구현

## 프로젝트 진행 방식
* 이벤트 스토밍의 영상을 흥미롭게 본 기억이 있어 프로젝트에 적용해보여 진행하였습니다.
